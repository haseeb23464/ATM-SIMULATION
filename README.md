ATM Machine Simulation (Python)

##  Project Description
This is a simple ATM Machine Simulation program written in Python.  
It allows the user to:

- Check account balance
- Withdraw money
- Deposit money
- Exit the system

The program uses basic Python concepts such as:
- Variables
- If-else statements
- Loops
- User input
- Arithmetic operations

---

##  Features

PIN authentication  
 Balance checking  
 Money withdrawal (with insufficient balance check)  
 Money deposit  
 Continuous menu system using while loop

##  Technologies Used

- Python 3
